# 数据库系统概论大作业

## 记录管理模块

### 文件

一张表使用一个文件存储。

文件的首页存储表的元数据，依次约定为：

| 项目     | 大小（字节） | 数量     | 前缀和（字节） |
| -------- | ------------ | -------- | -------------- |
| 属性数   | 1            | 1        | 1              |
| 属性信息 | 128          | $\le 32$ | 4097           |
|          |              |          |                |
| 记录条数 | 4            | 1        | 5011           |
