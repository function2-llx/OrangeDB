# 数据库系统概论大作业

## 使用方法

### 构建

依赖：

- [CMake](https://cmake.org/)

使用如下命令行构建数据库管理系统：

```bash
> cd dbms
> mkdir build
> cd build
> cmake ..
> make
```

## 记录管理模块

### 文件

一张表使用一个文件存储。

文件的首页存储表的元数据，依次约定为：

| 项目     | 大小（字节） | 数量     |
| -------- | ------------ | -------- |
| 属性数   | 1            | 1        |
| 属性信息 | 不定         | $\le 32$ |
| 记录大小 | 2            | 1        |
| 记录条数 | 4            | 1        |
